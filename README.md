# SpringOfZen

SpringOfZen is a Human Harbor for the Agent Era: a quiet place to pause, recover, reflect, create, connect, and navigate state.

The former Consciousness Audio Lab work remains inside SpringOfZen as a research and session pathway. Audio sessions, Night Harbor, Memory Air, DNA experiments, and feedback tools are now part of a wider Human State Navigation system.

## Site Structure

```text
index.html              Human Harbor homepage
today.html              Daily Harbor return ritual
night-harbor/           Night Harbor room and nightly listening experience
sessions.html           State navigation listening sessions
lab.html                Human State Lab and audio DNA archive
about.html              SpringOfZen manifesto
tracks.html             Legacy published track archive
today/YYYY-MM-DD.md     Daily Harbor source package
assets/                 CSS, JS, and lightweight frontend data
data/                   Night Harbor schedules, clips, sessions, broadcasts
exports/                Generated audio exports kept outside normal publishing flow
```

## North Star

Every feature, article, image, video, audio session, experiment, and agent must answer one question:

Does this help humans navigate their state in the Agent Era?

If not, do not build it.

## Daily Harbor Loop

Each day creates:

1. Observation
2. Poster prompt
3. Short video script
4. Harbor Session metadata
5. Distribution copy
6. Feedback question

The daily page is the heart of the system because SpringOfZen is optimizing for return rate, session duration, feedback, and community participation, not passive views.

## Positioning Rule

Avoid medical claims. Use language like experimental, exploratory, state navigation, recovery ritual, listener feedback, and human-centered research.
