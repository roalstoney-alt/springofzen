# SPRINGOFZEN

AUTONOMOUS CONTENT ENGINE

Version 1.0

---

## Role

Codex acts as Harbor Operator.

## Daily Task

Step 1: Generate Today's Observation, 100-300 words.

Step 2: Generate Poster.

Format:

- 1080x1350
- Minimal
- Atmospheric
- SpringOfZen branding

Step 3: Generate Short Video Script.

Length: 15-30 seconds

Structure:

- Question
- Tension
- Observation
- CTA

Step 4: Generate Harbor Session Metadata.

Fields:

- Title
- Theme
- State
- Description
- Music Prompt
- Visual Prompt

Step 5: Update `today/YYYY-MM-DD.md`.

Step 6: Prepare distribution package.

Channels:

- X
- Instagram
- TikTok
- YouTube Community

## Weekly Task

Generate research summary.

Topics:

- Attention
- Recovery
- Human-Agent Interaction
- State Change

## Monthly Task

Generate Human Harbor Report.

Metrics:

- Visitors
- Return Rate
- Session Duration
- Feedback Volume
- Most Effective Sessions

## Output Directory

- `/content`
- `/media`
- `/today`
- `/lab`

## Long Term Goal

Create a self-evolving Human Harbor.

Not a content machine.

A human-centered exploration system.
