# SPRINGOFZEN

90-DAY OPERATING SYSTEM

Version 1.0

---

## Mission

Build the first Human Harbor.

## Primary Objective

Find the first 100 people who repeatedly return.

## Success Metrics

Not:

- Followers
- Likes
- Views

Yes:

- Return Rate
- Session Duration
- Feedback Count
- Community Participation

## Content Structure

Daily:

1. Observation
2. Poster
3. Short Video
4. Harbor Session

Weekly:

1. 2 Essays
2. 1 Research Note

Monthly:

1. 1 State Report

## Month 1

Theme: Agent Era

Questions:

- What changes when AI becomes abundant?
- What remains uniquely human?
- What is attention worth?

## Month 2

Theme: State Navigation

Questions:

- What is a state?
- What changes a state?
- What restores a state?

## Month 3

Theme: Human Harbor

Questions:

- Why do people need harbors?
- What makes people stay?
- What creates meaning?

## Distribution

X: Daily observations

YouTube: Night Harbor and long-form sessions

TikTok: State clips, 15-30 seconds

Medium: Long-form essays, 2 per week

GitHub: Research log and public experiment

## Feedback Loop

User -> Session -> Feedback -> Research -> Improvement

Repeat forever.
